# Car Rental System

- A car Rental System made in HTML, CSS, JavaScript, Boostrap3, AJax, and PHP with MySQL.

## Core Features
- Registration system
- customer management system
- Checkout Details
- Reviews
- Payment integrated system using Stripe Api and Paypal
- Admin Site to manage client site.
